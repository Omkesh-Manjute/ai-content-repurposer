# How to run this project on another computer (Hindi/English Guide)

Es project ko dusare PC mein chalane ke liye niche diye gaye steps follow karein:

### 1. Prerequisite Softwares (Pehle ye install karein)
*   **Node.js:** Download and install (v18 or higher) from [nodejs.org](https://nodejs.org/).
*   **Python:** Install Python 3.10+ from [python.org](https://www.python.org/). (Installation ke waqt "Add Python to PATH" tick karna na bhulein).
*   **FFmpeg:** Ye video clipping ke liye zaroori hai.
    *   Download karein: [ffmpeg.org](https://ffmpeg.org/download.html)
    *   Isko extract karke iske `bin` folder ka path system **Environment Variables (PATH)** mein add karein. (Agar nahi hota toh batana, main detail step de dunga).

### 2. Project Setup
Folder ko naye PC mein copy karein aur Terminal/CMD open karke ye commands chalayein:

```bash
# 1. Frontend dependencies install karein
npm install

# 2. Video clipping tool install karein
pip install yt-dlp
```

### 3. Running the App
Sab kuch setup hone ke baad, bas ye command chalayein:

```bash
npm run dev
```
Ab browser mein `http://localhost:3000` open karein.

---

### Important Notes:
1. **API Keys:** App chalne ke baad aapko Gemini ya Groq ki API key UI mein daalni hogi.
2. **FFmpeg Error:** Agar video clip nahi ho rahi, toh check karein ki FFmpeg correctly installed hai ya nahi (CMD mein `ffmpeg -version` chala kar check karein).
